# TrueHost-Web-Intelligence-System
TrueHost Web Intelligence System is an OSINT-based tool that uncovers the real hosting origin of suspicious websites hidden behind CDNs. It analyzes DNS, WHOIS, CT logs, and subdomains to identify possible origin servers. The system provides fast, accurate insights for cybersecurity and forensic investigations.
